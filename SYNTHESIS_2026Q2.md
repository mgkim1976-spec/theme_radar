## 1. 메타 인사이트 — 시스템 자체에서 배운 것

### 1.1 시기 효과의 함정 (가장 큰 발견)

raw return 으로 채널을 비교하면 **잘못된 결론**에 도달한다:

| 채널 | 7d raw | 7d α | 90d α |
|---|---:|---:|---:|
| supergaemi | +7.65% | +0.96% | **−9.51%** |
| seo_jaehyung | +2.75% | +0.59% | +12.27% |
| 86bunga | +0.13% | −0.74% | +1.70% |

**supergaemi 는 단기 raw +7.65% 로 압도적 1위로 보였지만, 90일 α 는 −9.5%** — 시장보다 못함. 단기 차트·수급 신호가 30일+ 에 가면 시장 추세 못 따라잡고 noise 에 묻힘.

→ **단기 raw return 으로 채널 평가하면 안 됨**. index-relative α + 충분한 윈도우 (≥ 30d) 필수.

### 1.2 진짜 alpha generator — 강세장 ↔ 약세장 효과

**데이터 백필 후 (v2 갱신)**:

| 채널 | 90d α (v1, 강세장만) | 90d α (v2, 약세장 포함) | 변화 |
|---|---:|---:|---:|
| **seo_jaehyung** | +12.27% (n=381) | **+4.03%** (n=1,163) | **−8.2%p** ↓ |
| 86bunga | +1.70% (n=1,130) | +1.70% (변동 없음) | — |
| blueoak | +1.00% (n=854) | −1.08% (n=5,296) | −2.1%p ↓ |
| supergaemi | (n부족) | −9.51% (n=5,108) | (신규 측정) |

**핵심 인사이트**: 
- seo_jaehyung 의 +12% alpha 는 **강세장 (2024-2026) 효과 절반 포함**
- 약세장 (2023 H1) 시기 데이터 추가하니 +4% 로 축소
- 그래도 **5채널 중 1위** — 진짜 alpha 가 작지만 존재

**Overall α (전 5채널 통합)**: 30d −1.07%, 90d **−4.01%**, win 43%
→ *대부분 채널이 시장 평균보다 못함*. 직관과 반대.

### 1.3 데이터 기간/규모는 평가 신뢰도에 결정적

데이터 풍부함 → alpha 신뢰도 정확 측정의 사례:

| 채널 | 측정 단계별 90d α 변화 |
|---|---|
| seo_jaehyung | n=381 (+12.3%) → n=1,163 (+4.0%) |
| blueoak | n=854 (+1.0%) → n=5,296 (−1.1%) |
| supergaemi | (측정 불가) → n=5,108 (−9.5%) |

**규칙**: n < 1,000 신뢰 낮음. n ≥ 5,000 안정. n 늘어날수록 alpha 보수적으로 수렴.

### 2.2 86bunga — *약세장 검증된 가치투자*

**메소돌로지**: 거시→정책→공시→회계·거버넌스→가격 논리 번역

**Signal**: 거시지표(16.3%)·정책_규제·공시·수급·기술적

### 2.3 supergaemi — *단기 차트·수급 트레이더, alpha 음수*

**메소돌로지**: "실적+수급+차트" 3각, 공시로 모멘텀 선점

**Signal**: 수급_거래량(58.7%)·기술적_차트(50.7%) — 다른 채널 대비 +25%p 압도

**진단**: bottom-up 단기 트레이딩이 ground truth 면 30일+ 보유 hypothesis 와 안 맞음.

### 2.4 blueoak — *외국인 모니터, 정보가치 ≠ alpha*

**메소돌로지**: 수급-차트-기관 동시 확인, 외국인 흐름 1순위, 거시 top-down

## 6. 메타 메소돌로지 — 시스템이 학습한 것

### 6.1 어떤 신호 조합이 유효한가

데이터로 검증된 양의 α 패턴:
1. **정책_규제_해외 + 반도체 테마** → CHIPS Act·IRA 발 종목 (HBM·SK하이닉스 등)
2. **밸류에이션_분석 + 자사주_매입 event** → 주주환원 카탈리스트 + 저평가
3. **지배구조_변경 + 펀더멘털** → 회계·자본배분 변화 시점

### 6.2 어떤 신호 조합이 안 되는가

- **수급_거래량 + 기술적_차트** (한국 유튜브 dominant 패턴): 단기 noise, 30d+ 음수 α
- **신제품_출시 + 신사업_진출** ("성장 스토리"): 발표 후 underperform
- **M&A_지분**: 직관과 반대로 음수 (인수자 주주가 손해)

### 6.3 채널별 메소돌로지 효율 ranking

90d α × win_rate 결합 점수:
1. seo_jaehyung — *명확한 alpha* (산업 사이클)
2. 86bunga — *sustainable* (거버넌스·약세장)
3. blueoak — *flat* (외국인 모니터)
4. supergaemi — *negative* (단기 차트)
5. han_gyunsoo — *insufficient*

→ **차트·수급 위주 채널은 시장 베타 정도. 산업 사이클·펀더멘털 분석이 alpha**

## 9. 시스템 한계 및 다음 단계

### 한계
1. **Survivorship bias**: alpha 큰 테마 (반도체) 가 86bunga 시기엔 다른 모습이었을 수 있음
2. **시기 매칭 불완전**: 86bunga vs 다른 채널 직접 비교 불가 (시기 다름)
3. **Skewed return**: 일부 테마 (로보티즈 +26.4%) 는 outlier 효과
4. **메소돌로지 합성**: 채널 본인이 한 분석이 아니라 LLM 의 추론

### 다음 단계 (제안)
- **Regime-conditional alpha** — OVERHEATING/CONTRACTION 별 분리 측정
- **Phase tracking** — 사이클 어디인지에 따라 동일 신호의 α 다름 검증
- **Skew test** — outlier 1-2개 빼면 어떻게 되나 (robustness)
- **Decay test** — 발화 후 7d/30d/90d 시간축 따라 α 진화 (이미 부분 구현)

## 10. 한 문장 요약

> 한국 투자 유튜브 5채널 평가 결과, **차트·수급 위주 분석은 시장 평균 정도** (또는 음수 α), **산업 사이클 + 정책_규제_해외 조합** 이 진짜 alpha 의 유일 후보. **반도체·AI 인프라가 현 사이클의 leader**, **바이오·스테이블코인은 30일+ 보유 시 손실**.
>
> **v2 결정적 발견**: 강세장만 본 데이터로는 alpha 가 부풀려진다. 약세장 포함 후 진짜 alpha 는 **절대값 1-4% 수준** — 운용비·세금·거래비 차감하면 거의 0.
